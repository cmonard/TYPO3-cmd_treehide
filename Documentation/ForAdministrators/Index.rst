Administration
==============

Import
------

Import Extension from the TYPO3 Extension Repository to your server.


Install
-------

Install the extension.


Configuration
-------------

Go to your profile, in the "Edit & Advanced functions" tab, you should now find
a checkbox before the "Recursive Delete", named "Recursive tree hide". Just
check if you want to activate the functionnality.

|user-option|

Usage
-----

Backend: The tree not hidden yet
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

|tree-active|

The active tree we want to hide. We click on the "cmonard.com" page icon to activate the click menu


Backend: The activate menu, still the same
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

|tree-menu|

Now we will click on the "Disable" menu.


Backend: The tree is now hidden
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

|tree-inactive|

The entire tree is now hidden, all the subpage have been modified.


Backend: The tree is now hidden
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

|tree-inmenu|

By clicking on a subpage we see that the page is hidden and can be showed up again.
